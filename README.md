# opengl实现魔方
* 环境: 天空盒、光照、阴影、第三人称摄像机
* 操作: 按住鼠标中键移动鼠标旋转视角、滚动鼠标滚轮放大视角
* 魔方: 
  - 点击后会框选一层，鼠标左键框选水平一层，右键框选竖直一层
  - 上下左右键实现层的旋转
  - 为了模拟现实魔方效果，需要保证所有水平层对齐的情况下才能旋转列，反之同理。魔方可以容忍一定范围的不对齐并进行校正
  

编译环境: VisualStudio 2017
